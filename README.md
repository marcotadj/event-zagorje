# Event.zagorje — materijal za web stranicu

Mapa sa svim podacima povučenim s Instagrama **@event_zagorje** (30.06.2026.),
spremno za predaju **Claude Code-u** da izradi web stranicu.

## Kako koristiti
1. Otvori cijelu ovu mapu u Claude Code.
2. Reci mu: **„Pročitaj INSTRUKCIJE-ZA-CLAUDE-CODE.md i napravi stranicu."**
3. On će iskoristiti `podaci.json`, `SADRZAJ.md` i slike iz `images/`.

## Sadržaj mape
- **INSTRUKCIJE-ZA-CLAUDE-CODE.md** — glavni brief (što i kako napraviti) 👈 počni ovdje
- **SADRZAJ.md** — sav tekstualni sadržaj, čitljivo
- **podaci.json** — isti podaci strukturirano (za parsiranje)
- **images/** — profilna, 10 foto s evenata, 11 ikonica kategorija
- **_scrape_log/** — radne datoteke (izvorni URL-ovi); može se ignorirati
